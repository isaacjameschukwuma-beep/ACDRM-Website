# Screenshots — ACDRM Website

This folder stores screenshots of the website for review, tracking, and flagging areas to change or amend.

## How to use

- Drop screenshots here as you review pages (homepage, founder, contact, etc.).
- Suggested naming so they sort and read clearly:
  - `index-desktop.png`, `index-mobile.png`
  - `founder-desktop.png`, `founder-tablet.png`, `founder-mobile.png`
  - `contact-desktop.png`
  - Add a date or short note for revisions, e.g. `founder-hero-2026-06-07-spacing-fix.png`
- For something that needs changing, a short note next to the file (or in the filename) helps track the issue, e.g. `founder-works-FIX-titles.png`.

## Purpose

A single place to track the visual state of each page across breakpoints (1440 / 1024 / 820 / 480px) and to mark areas for amendment as the build progresses.
